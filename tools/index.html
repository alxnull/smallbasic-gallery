Tools

Gallery of developer tools for working with Small Basic. This includes IDEs, compilers, editors and more. These tools are usually not written in Small Basic themselves.

  • MySmallBasic by Kwanghoon ChoiIDE

    An open-source Small Basic Interpreter and Debugger written in Java.

  • SB IDE++ by gungan37IDE

    SB IDE++ is a replacement for the default Small Basic IDE that allows intermediate/advanced SB developers to use a more advanced IDE to boost productivity and to provide a stepping stone to Visual Studio and other full-featured IDEs.

  • Small Basic Desktop by MicrosoftIDE

    The official Small Basic Desktop IDE for Windows, developed by Microsoft.

  • Small Basic Online by MicrosoftIDE

    The next generation of Small Basic. A completely new online editor that can run and debug Small Basic programs in the browser.

  • Small Basic Prime by LitDevIDE

    An extended interface for Small Basic. It includes all the features present in the standard Small Basic IDE, with some additional features. Main additional features are debugger, integrated extension manager and updated text editing.
