# Welcome to the Small Basic Gallery!

Small Basic is a very simple programming language, ideal for those learning to code.
Check out the official Small Basic website at [smallbasic.com](http://smallbasic.com) to get started!

This gallery is a collection of programs, extensions and tools made by and for the community of Small Basic. It showcases what everyone can achieve with programming in Small Basic.

This website is built from [a GitHub repository](https://github.com/alxnull/smallbasic-gallery). Contribute and add your own software! ([learn more](https://alxnull.github.io/smallbasic-gallery/contribute))
